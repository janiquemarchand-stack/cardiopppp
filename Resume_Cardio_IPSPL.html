TVP · Embolie pulmonaire
TVP
Thrombose veineuse profonde
Triade de Virchow · Score Wells TVP
Sx +
  • Douleur MI (mollet)
  • Érythème · chaleur
  • Œdème unilatéral
  • Cordon veineux palpable
  • Différence mollet >3 cm
DDx −
  • Rupture gastrocnémien
  • Kyste de Baker rompu
  • Cellulite
  • Insuffisance veineuse
  • Traumatisme musculaire
Algorithme Wells TVP
  • Wells <2 (improbable) → D-dimères → si − : pas TVP · si + : Doppler MI
  • Wells ≥2 (probable) → Doppler veineux MI directement
  • D-dimères : haute sensibilité, faible spécificité → utile seulement si Wells <2
  • Si délai >4h → initier anticoagulation avant résultat (sauf CI saignement)
Traitement
  • AOD (majorité) · HBPM + Warfarine · HBPM monotx si cancer/grossesse
  • Durée minimale 3 mois → réévaluation selon cause / FdR récidive

DLP
Dyslipidémie et physiopatho athérosclérose
Score Framingham · Cibles LDL
Physiopathologie plaque (résumé)
  • Lésion endothéliale → LDL s'infiltrent → oxydation → macrophages = foam cells → noyau lipidique → plaque
  • Plaque ≥75% lumière → ischémie transitoire (angine) · Rupture plaque → thrombus → SCA
  • Plaque vulnérable : gros noyau lipidique · recouvrement mince · inflammation active
Dépistage (groupe PEER)
  • H ≥40 ans · F ≥50 ans (plus tôt si DB, HTA, tabac, obésité, MCAS fam, néphropathie chronique)
  • Bilan lipidique : CT · LDL · HDL · TG · Non-HDL · (ApoB si TG ≥1.5 — SCC)
Cibles selon risque (SCC)
  • Élevé (SRF ≥20%) → Statine forte/haute · Cible LDL <2 mmol/L ou réduction ≥50%
  • Modéré (SRF 10-19%) → Statine si LDL ≤3.5 · Cible LDL <2 mmol/L
  • Prévention secondaire (MCVAS) → Statine max dose · LDL <1.8 mmol/L
  • HF (LDL ≥5 si ≥40 ans) → Statine + iPCSK9 · Réf génétique
Statine CI grossesse Pré-éclampsie = FdR CV futur (RR 2.3) ApoB si TG ≥1.5 (plutôt que LDL)

Physiopathologie en 3 lignes — questions d'examen fréquentes
PHYSIO
Pourquoi dlr irradie au bras/mâchoire ?
  • Récepteurs nociceptifs myocardiques → influx via C8-T1 à T4
  • Tractus ascendant partagé avec mâchoire/cou/épaules/bras → douleur référée
  • Même dermatome que les bras → souvent bilatéral (plus fréquent)
PHYSIO
Pourquoi le BNP est élevé dans l'IC ?
  • Surcharge volumique → étirement des myocytes ventriculaires
  • BNP = peptide natriurétique libéré en réponse → vasodilatation + natriurèse compensatrices
  • Faux − : obésité · Faux + : MPOC sévère · Entresto
PHYSIO
Pourquoi FA → risque d'AVC ?
  • Activation désorganisée (350-600 btm) → perte du kick auriculaire
  • Stagnation sang oreillette G (auricule) → thrombus → embolisation cérébrale
  • Anticoagulation vise à prévenir ce thrombus (pas l'arythmie elle-même)
PHYSIO
Pourquoi sevrage BB → rebond HTA/tachycardie ?
  • Sous BB chronique → surexpression des récepteurs β par compensation
  • Arrêt brutal → levée du frein sympathique → hyperstimulation β1 → ↑FC ↑TA ↑contractilité
  • Sevrage graduel recommandé · Délai d'apparition : 24-72h
PHYSIO
Pourquoi IECA contre-indiqué si sténose rénale ?
  • Sténose → ↓pression perfusion → ↑Ang II → vasoconstriction efférente → maintien DFG
  • IECA bloque Ang II → dilatation efférente → ↓pression de filtration → créat ↑ >30%
  • Signe d'alerte HTA secondaire rénovasculaire
PHYSIO
Différence EP vs TVP : physiopatho
  • TVP : thrombus veineux → stase + hypercoagulabilité + lésion endothéliale (Virchow)
  • EP : embole migre vers artère pulmonaire → ↑espace mort alvéolaire → bronchoconstriction → ↑PAP → défaillance VD
  • 10% EP fatals dans l'heure · 25-50% EP ont sx de TVP concurrent
